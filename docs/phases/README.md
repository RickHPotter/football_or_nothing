Football or Nothing - Phase Documentation
=========================================

Purpose
-------
This folder records what each implemented phase actually added to the game.

The planning folder remains the roadmap. These phase documents are the working
implementation reference: which models, services, screens, and behaviors exist
now, what was verified, and what was deliberately deferred.
