Football or Nothing - First Playable Backlog
============================================

Goal
----
Reach a minimal playable football-management loop before expanding the world.


Phase 1 - Foundation
--------------------
Status: mostly implemented.

1. Add default Rails authentication.
2. Create the user-to-manager career concept.
3. Create countries.
4. Create clubs.
5. Create initial club finances.
6. Create stadiums.
7. Create athletes.
8. Create athlete attributes.
9. Create athlete contracts.
10. Generate a tiny fictional world:
   - 1 country
   - 8 clubs
   - 8 stadiums
   - 18 to 25 athletes per club
   - Basic financial profiles for each club
   - 1 manager controlled by the user


Phase 2 - Club Management Skeleton
----------------------------------
Status: mostly implemented.

1. Show manager dashboard.
2. Show current club dashboard.
3. Show squad list.
4. Show athlete profile.
5. Show contracts and wages.
6. Allow manager to choose or be assigned to a club.
7. Show club financial summary.
8. Prevent the player from acting as club owner.


Phase 3 - Tournament Skeleton
-----------------------------
Status: implemented for a generated domestic league.

1. Create tournament.
2. Create tournament edition.
3. Register clubs through tournament participations.
4. Generate fixtures for a simple league format.
5. Store standings.
6. Mark champion at season end.


Phase 4 - Match Simulation v1
-----------------------------
Status: implemented.

Implemented:
- Fixture page.
- Deterministic instant simulation.
- Match status and score storage.
- Standings updates.
- Match timeline with goal events.
- Career date advancement to match day and next match.
- Basic athlete stats from simulated matches.
- Lineups and bench selections.
- Live clock in 15-minute advances.
- Pausing and resuming for manager decisions.
- Substitutions.
- Basic tactical changes through formation and mentality.

Deferred:
- True minute-by-minute simulation events.
- Detailed events beyond goals.


Phase 5 - History
-----------------
Status: core implemented.

Implemented:
- Store trophies.
- Store tournament participation outcomes.
- Complete tournament editions when all fixtures are completed.
- Assign champion from standings.
- Store manager season stats.
- Store club season stats.
- Store athlete season stats.
- Show trophy history on career and club pages.
- Show manager season history on career page.
- Show club season history and top scorers on club page.
- Show athlete stats on athlete profiles.

Deferred:
- Top scorer awards.
- Best player/keeper/young player awards.
- News/history feed.
- Hall of fame records.
- Promotion/relegation history.
- Rich financial season snapshots.


Phase 6 - Transfer Skeleton
---------------------------
1. Create transfers as historical records.
2. Support free transfer.
3. Support permanent transfer.
4. Update athlete contracts through transfer flow.
5. Keep old contracts/history intact.
6. Enforce basic transfer budget and wage budget rules.


Phase 7 - RPG Progression
-------------------------
1. Add manager reputation.
2. Add athlete attributes and potential.
3. Add aging.
4. Add morale and condition.
5. Add club reputation.
6. Use manager reputation to limit available jobs.


Important Deferrals
-------------------
These are important, but not required for the first playable loop:

- Complex tactics.
- Detailed match events.
- Full staff model.
- Youth academy.
- Scouting.
- Sponsorships.
- Stadium construction and upgrades.
- International tournaments.
- National teams.
- Multi-country calendar.
- Advanced financial rules.
- Press/news system.
- Save import/export.
- Real data import.
